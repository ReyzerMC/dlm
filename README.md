# Just a Rust Download Manager ⚙️

**DLM** is a super simple download manager writed in Rust

---

## Features

- Download things
- Save the output in a specified file

---

## Instalation

1. Clone this repository

```bash
git clone https://github.com/ReyzerMC/dlm.git
cd dlm
```
Dependencies:

`cargo` to compile

---

2. Compile the project with cargo

```bash
cargo clean && cargo build --release # Output is at ./target/release/dlm
```

---

## Usage:

Run `dlm -h / --help` (if installed on /bin) or `./dlm -h / --help`
